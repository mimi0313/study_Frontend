## HTML basic
- HTML : Hyper Text Markup Language : 웹 페이지를 표시하는 언어
  - Hyper Text : 하이퍼링크로 연결되어 있는 문서 => 웹 페이지(콘텐츠,구조)
  - Markup : 표시 
  - Language : 언어

- HTML 문법
  - 명칭 : Tag, Element
  - 구성 : 시작태그 ~ 종료태그
  - 종료태그가 없는 태그 : 빈태그 (Empty Element)
```
  <tag> contents </tag> : Element

  <tag ...> : Empty Element
```
